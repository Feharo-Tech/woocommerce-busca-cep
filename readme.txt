Permite aos clientes buscar o CEP preenchendo o endereço no checkout do WooCommerce.

== Description ==

O plugin **WooCommerce Busca CEP por Endereço** adiciona uma funcionalidade prática ao seu checkout, permitindo que clientes encontrem seu CEP digitando o endereço.

= Principais recursos =
* Busca de CEP por endereço (logradouro, cidade e UF)
* Integração perfeita com o checkout do WooCommerce
* Preenchimento automático dos campos de endereço
* Suporte a múltiplos resultados de CEP
* Shortcode para usar em qualquer página
* Totalmente personalizável:
  * Texto do link
  * Posição do link
  * Cores personalizadas
* Tradução pronta para internacionalização

== Installation ==

1. Envie o arquivo ZIP através do menu 'Plugins > Adicionar Novo' no WordPress
2. Ative o plugin
3. Configure as opções em 'WooCommerce > Busca CEP'
4. O recurso estará automaticamente disponível no checkout

== Frequently Asked Questions ==

= O plugin funciona com qualquer tema? =
Sim, o plugin foi testado com os principais temas do WooCommerce incluindo Storefront, Astra e OceanWP.

= Posso usar o formulário de busca em outras páginas? =
Sim, basta usar o shortcode `[busca_cep]` em qualquer página ou widget.

= A API de busca de CEP tem limite de requisições? =
Sim, a API ViaCEP usada pelo plugin tem limites de requisições. Para uso comercial intensivo, considere usar uma API paga.
